# Árvore AVL em Java

# Como compilar e executar o projeto (Árvore AVL)


## Compilar (Linux / macOS / Windows)
cd src
javac No.java ArvoreAVL.java Main.java Validador.java

## Para executar - com entrada via arquivo (testes)
java Main < ../tests/teste1.in > saida1.txt


## Compare a saída com o arquivo esperado:

### Linux / macOS
diff -u ../tests/teste1.out saida1.txt

### Windows
fc saida1.txt ..\tests\teste1.out
